## Overview 
Z&R Car Rental System is a full-stack platform designed for car rental companies. It integrates a robust database backend, user-friendly front-end interfaces, and reporting tools to manage customers, vehicles, bookings, and administrative tasks efficiently. The system automates core operations and enhances the overall customer experience.



## Key Features
- **Customer Interface:** Registration, login, profile management, booking vehicles 
- **Admin/Manager Interface:** Manage vehicles, monitor bookings, update inventory, generate reports   
- **Vehicle Filtering:** Search cars by brand, color, type, rental cost, or features 
- **Booking System:** Select rental dates, check availability, and calculate total cost 
- **Reports:** Cars, Customers, Bookings, Car Brands using JasperReports  
- **Security:** Unique user accounts, secure login, and data integrity   
- **Full-stack Implementation:** Database + Back-end + Front-end GUI  

## Technical Highlights 
- **Database:** PostgreSQL via PgAdmin 4, fully normalized (BCNF compliant)  
- **Backend:** Java application logic implemented with NetBeans  
- **Frontend/GUI:** Interactive GUI for customers and managers  
- **Reporting:** Jaspersoft Studio for charts, tables, and analytics  
- **Modeling:** UML Class Diagram, EER diagram, functional dependencies analysis

 ## Learning Outcomes
- Understand full-stack development combining database, backend logic, and GUI
- Implement secure booking and user management systems
- Generate dynamic reports for operational and business insights
